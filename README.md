# rhel10-monitoring-stack
Скрипт на Python для автоматического развертывания стека мониторинга (**Prometheus, Grafana, Node Exporter, Process Exporter**) на операционной системе Red Hat Enterprise Linux 10 (RHEL 10) с использованием rootless-контейнеров **Podman**.

## 🚀 Особенности скрипта
* **Полная адаптация под RHEL 10:** Настраивает доверенные реестры, активирует пользовательский сокет `podman.socket` для совместимости с `docker-compose`.
* **Решение проблем с SELinux:** Конфигурирует безопасное чтение `/proc` хоста для `process-exporter` без нарушения политик безопасности системы.
* **Интерактивность:** Позволяет выбрать только нужные компоненты при запуске.

## 📋 Системные требования
* Операционная система: **RHEL 10** или **CentOS Stream 10**
* Установленный **Python 3**
* Доступ в интернет (для скачивания образов)

🗂 Структура генерируемых файлов
Скрипт автоматически создаст в директории запуска:

docker-compose.yml — конфигурация манифеста для Podman Compose.

config/prometheus.yml — преднастроенные задачи (jobs) сбора метрик.

data/ — постоянные хранилища (volumes) для баз данных Prometheus и Grafana.
